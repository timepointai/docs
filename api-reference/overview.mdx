# API Overview

The Timepoint ecosystem exposes three HTTP APIs. The Clockchain public endpoints require no authentication. Flash and Pro require service keys.

## Base URLs

| Service | Base URL | Auth Required |
|---------|----------|---------------|
| **Flash** | `https://api.timepointai.com` | Yes (`X-Service-Key`) |
| **Flash** (alias) | `https://flash.timepointai.com` | Yes (`X-Service-Key`) |
| **Clockchain** (public) | `https://clockchain.timepointai.com` | No |
| **Clockchain** (write) | Same as above | Yes (`X-Service-Key`) |
| **Pro Cloud** | `https://pro.timepointai.com` | Yes (`PRO_API_KEY` or Flash JWT) |
| **MCP** | `https://mcp.timepointai.com` | Yes (MCP API Key) |

## Quick Examples

### Clockchain — No Auth Needed

```bash
# Get graph stats
curl https://clockchain.timepointai.com/api/v1/stats

# Search moments
curl "https://clockchain.timepointai.com/api/v1/moments?q=rome&limit=5"
```

### Flash — Render a Moment

```bash
curl -X POST https://api.timepointai.com/api/v1/timepoints/generate/sync \
  -H "Content-Type: application/json" \
  -H "X-Service-Key: YOUR_KEY" \
  -d '{"query": "Moon landing, July 20 1969", "generate_image": true}'
```

## Response Format

All APIs return JSON. Clockchain public endpoints return paginated results:

```json
{
  "items": [...],
  "total": 769,
  "limit": 10,
  "offset": 0
}
```

## Rate Limits

| Tier | Limit | Applies To |
|------|-------|------------|
| Public | 60/min | Clockchain unauthenticated reads |
| Auth reads | 300/min | Authenticated GET requests |
| Auth writes | 30/min | POST/PUT/DELETE with service key |

Rate limit headers: `X-RateLimit-Limit`, `X-RateLimit-Remaining`, `X-RateLimit-Reset`

## CORS

All backend services allow cross-origin requests from `*.timepointai.com` subdomains. Each service also accepts a `CORS_ORIGINS` env var for additional origins.

| Service | Allowed Origins | Methods |
|---------|----------------|---------|
| Flash | `timepointai.com`, `app.`, `pro.`, `clockchain.`, `flash.`, `timepoint.ai` | All |
| Clockchain | `timepointai.com`, `app.`, `api.`, `flash.`, `localhost:3000` | GET, OPTIONS |
| Pro Cloud | `timepointai.com`, `app.`, `api.`, `flash.`, `timepoint.ai`, `localhost:3000` | All |
